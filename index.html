Assignment 1

Description

The purpose of this assignment is to learn the basics of HTML to create a simple "About Me" page. This page includes a few paragraphs about myself, a list of my favorite websites, my class schedule in table format, and a picture of me. I used CSS and Bootstrap to make it look pretty.

During this assignment, I learned how to create tables and neatly organize my folder structure. I have coincidentally already done most of this assignment as a personal project over the summer so a lot of my HTML was repurposed, but this was a good refresher.

Objectives:

  • Learn the fundamentals of web design and HTML
  • Utilize appropriate HTML tags
  • Learn proper formatting and layout
  • Become familiar with the development process

Assignment 2

Description

In this assignment, I demonstrate decision making and web forms by creating a page that collects user information and decides what to do with it. Feedback is returned depending on the data given.

During this assignment, I learned how to create various web form elements such as radio buttons and text boxes, as well as how to write decision control structures with Javascript.

Objectives:

  • Learn about web forms
  • Implement decision making
  • Create flowcharts in Visio

Assignment 4

Description

Objectives:

Assignment 6

Description

Create a movie backlog application that performs MySQL queries. The user can add, modify, and delete records in the backlog. The backlog contains info pulled from the MySQL database.

I learned how to create and modify tables in MySQL and execute queries in PHP.

Objectives:

  • Learn to execute queries in PHP
  • Get familiar with MySQL
  • Design database tables

Description

I learned how to use AJAX to add even more interactivy to my application.

Objectives:

  • Expand on previous application
